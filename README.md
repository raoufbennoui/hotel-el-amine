A complete multi-page hotel website built for a real 
hospitality business in Boumerdès, Algeria.

## Live Demo
[*(https://hotel-el-amine.netlify.app)*](https://hotelelamine.netlify.app/)

##  Preview
![Hotel El Amine](preview.png)

##  Features

- **Parallax hero section** with smooth scroll effect
- **Photo sliders** on room cards with fade transitions
- **Interactive gallery** with fullscreen lightbox viewer
- **Scroll animations** using Intersection Observer API
- **WhatsApp booking integration** — direct reservation system
- **Google Maps embed** for location
- **Mobile-first responsive design** — tested on all devices
- **Sticky navbar** with scroll-triggered style change
- **Multi-page architecture** — 6 separate pages

##  Pages

| Page | Description |
|------|-------------|
| `index.html` | Home — hero, rooms preview, restaurant, gallery, contact |
| `chambres.html` | Rooms — full room details with photo sliders |
| `restaurant.html` | Restaurant — menu, hours, ambiance photos |
| `galerie.html` | Gallery — masonry grid with filter tabs and lightbox |
| `a-propos.html` | About — hotel story, team, stats |
| `contact.html` | Contact — reservation form, map, WhatsApp |

##  Tech Stack

- **HTML5** — semantic markup
- **Tailwind CSS** — utility-first styling via CDN
- **Vanilla JavaScript** — animations, sliders, lightbox
- **CSS Custom Properties** — design system variables
- **Intersection Observer API** — scroll animations
- **Netlify** — deployment and hosting

##  Getting Started

No build tools required. Just open in your browser:
```bash
# Clone the repository
git clone https://github.com/raoufbennoui/hotel-el-amine

# Open in browser
open index.html
```

Or simply visit the **[live demo](https://hotel-el-amine.netlify.app)**

##  Business Context

Most hotels in Boumerdès, Algeria have no web presence 
or outdated websites that don't work on mobile. This 
project addresses that gap — giving Hotel El Amine a 
modern, professional online presence that converts 
visitors into guests.
